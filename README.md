## :clipboard: Descripción:
Proyecto web fullstack orientado a la gestión y visualización de productos de cerámica decorativa. La aplicación permite a los usuarios registrarse, autenticarse y navegar por un catálogo de productos, con una arquitectura separada entre backend y frontend que facilita la escalabilidad y el mantenimiento.

## :red_circle: Problema que soluciona

La empresa “Mundo de Mery” tiene dificultades en su operación diaria debido a que el manejo de inventarios y pedidos se realiza de forma manual. Provocando desorden, posibles errores en los registros y una gestión poco eficiente del negocio.

Además, la falta de presencia digital hace que sea más difícil llegar a nuevos clientes y aprovechar oportunidades de venta, limitando el crecimiento del emprendimiento.

## :diamonds: Características: 
* Autenticación de usuarios (registro y login).
* Seguridad JWT para contraseñas.
* Gestión de roles de usuario (usuario y admin).
* Catálogo de productos.
* Interfaz moderna.
* Comunicación entre frontend y backend mediante API REST.

## :computer: Tecnologías usadas: 
### Backend:
* JWT (JSON Web Tokens)
* Java
* Maven 
* Spring Boot
* MySQL

### Frontend: 
* Next.js
* React
* Tailwind 

## :star: Funcionalidades principales
* Registro e inicio de sesión de usuarios
* Autenticación segura mediante JWT
* Gestión de roles de usuario
* Visualización de catálogo de productos
* Organización de productos por categorías
* Consulta de detalles de cada producto
* Interacción con API REST (frontend y backend)
* Interfaz web moderna y responsiva
